# Roku Remote Control

![Roku Remote Control Screenshot](images/roku-remote-control-screenshot.png)

## Overview

A lightweight, single-page web application for controlling Roku devices via IP. This simple interface allows users to navigate and control their Roku devices seamlessly.

## Features

- Easy-to-use interface for controlling Roku.
- Customizable IP address for your Roku device.
- Supports navigation commands (up, down, left, right, select, home, back).

## Getting Started
